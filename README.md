# WidgetFlags-uScript

**A uScript module that allows you to hide and show widget flags**

## Flags
- None
- Modal
- NoBlur
- ForceBlur
- ShowInteractWithEnemy
- ShowDeathMenu
- ShowHealth
- ShowFood
- ShowWater
- ShowVirus
- ShowStamina
- ShowOxygen
- ShowStatusIcons
- ShowUseableGunStatus
- ShowVehicleStatus
- ShowLifeMeters
- Default
